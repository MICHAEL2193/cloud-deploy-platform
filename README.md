# Cloud Deploy Platform

Proyecto orientado a portafolio para perfil Cloud Engineer.  
La idea es desplegar una aplicación Node.js + Express en AWS siguiendo buenas prácticas básicas de arquitectura, disponibilidad, seguridad y evidencias técnicas.

## Objetivo

Construir una pequeña plataforma web desplegable en AWS, documentada de forma profesional en GitHub, mostrando:

- desarrollo local con Node.js + Express
- estructura de proyecto limpia
- evidencias del despliegue
- base preparada para escalar hacia una arquitectura cloud

## Stack

- Node.js
- Express
- Git + GitHub
- WSL2
- AWS
